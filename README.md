# 🌍 Global Multilingual Quiz Game

A production-ready, high-performance cross-platform Quiz Application built using **Flutter CLI**. This app targets a global audience by offering dynamic quiz challenges with an English user interface, while allowing users to play questions in their preferred local language.

## 🚀 Key Features

*   **Global Content:** Diverse question bank covering world history, geography, science, sports, and general knowledge.
*   **Multilingual Support:** Play quizzes seamlessly in **English, Urdu (اردو), Hindi (हिंदी), and Arabic (العربية)**.
*   **Smart UI & RTL Engine:** Automatically detects language and switches layouts to Right-to-Left (RTL) for Urdu and Arabic to ensure a flawless user experience.
*   **Lightweight & Scalable:** Structured using clean architecture principles, making it easy to scale the question bank to thousands of entries without performance lag.
*   **Monetization Ready:** Built-in structural architecture optimized for Rewarded and Banner Ads.

## 🛠️ Technical Specifications

*   **Framework:** Flutter (Dart)
*   **Package Name:** com.hamidabbas.quiz_game
*   **Development Environment:** Termux + Ubuntu (Root Container)
*   **Target Platforms:** Android (.aab ready for Google Play Store)
